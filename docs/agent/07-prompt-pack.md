# Prompt Pack（Claude Code / Codex 通用）

## 1. 新功能实现 Prompt
```text
你是仓库内的编程 Agent。请完成任务：<任务ID/标题>。
约束：
1) 先阅读相关文件并列出计划；
2) 先写失败测试再实现；
3) 最小改动，不重构无关模块；
4) 运行指定测试并汇报结果；
5) 输出变更文件和风险。
验收标准：<复制任务卡验收项>
```

## 2. Bug 修复 Prompt
```text
请按“复现 -> 定位 -> 修复 -> 验证”流程修复问题：<问题描述>。
必须先提供复现命令和失败证据，再改代码。
修复后运行：
- pytest tests/<相关测试>.py -v
- pytest -v
并报告根因、修复点、回归风险。
```

## 3. 代码评审 Prompt
```text
请对本次改动做代码评审，重点找：
1) 行为回归；2) 边界条件遗漏；3) 测试缺口；4) 风险场景。
输出按严重级别排序，包含文件路径和行号。
```

## 4. 文档同步 Prompt
```text
请检查本次改动是否需要更新 README / docs/plans / docs/agent。
若需要，直接补齐并给出变更摘要。
```

## 5. 发布前检查 Prompt
```text
请执行发布前检查：
1) 功能冒烟；2) 全量测试；3) 配置检查；4) 回滚路径检查。
输出 Checklist 结果和阻塞项。
```
